HTML

  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • width: 80%; /*80% of the available width*/
  • margin: 40px auto; /*40px to the top and bottom, and the auto value to the left and right.*/
  • border: 5px solid blue; /*border with a width of 5px that is solid and blue (originally gray)*/
  • box-shadow: 5px 10px #888888; /*adds shadow effects around an element's frame. The X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); The Y offset is 10px and defines the extent on the Y axis (or just below the box) A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. You can set multiple effects separated by commas. We haven’t defined the blur or spread radius here.*/

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge any commits from the tracking remote branch
  • git add [file]: add a file as it looks now to your next commit (stage)
  • git commit
  • git push
  • git merge

JavaScript

  • A variable is a named container that allows us to store data in our code: var shape = "circle";
  • Variables can contain strings: var topics = "HTML, CSS, Git, JavaScript";
  • Control flow is the order in which a computer executes code in a script.
  • If statements are a type of conditional statement. A tool to add logict to your code, that allow us to interrupt control flow.
  • An array is a single variable that is used to hold a group of data: var shapes = ["triangle", "square", "pentagon", "circle"]; To adress a specific item within the array we must adress the corresponting index value i.e. shapes[0]; = "triangle", shapes[1]; = "square", etc.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over e.g. for(var x=0; x < shapes.length; x++) {console.log(shapes[x]);}
  • Use the console.log() method to output a message to the web console, by adding an argument between the parenthesis.
  • function helloWorld() {}: A function is a set of instructions (between the curly brackets) that tells the computer how to perform a certain task. A function must be called to be executed i.e. helloWorld()
